# Decimal to Binary Info

- Decimal -> Binary -> Number of bits in binary -> `log_2(Decimal)`
    - `0` -> `0` -> 1 -> Not defined
    - `1` -> `1` -> 1 -> 0
    - `2` -> `10` -> 2 -> 1
    - `3` -> `11` -> 2 -> ~1.58
    - `4` -> `100` -> 3 -> 2
    - `5` -> `101` -> 3 -> ~2.32
    - `6` -> `110` -> 3 -> ~2.58
    - `7` -> `111` -> 3 -> ~2.81
    - `8` -> `1000` -> 4 -> 3

- Ceiling vs Floor:
    - ceil(2.9) = 3
    - floor(2.9) = 2
